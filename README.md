## 概要
私のWeb開発スキルと制作実績をまとめたポートフォリオサイトです。
教育・サービス業で培った「顧客視点」を活かし、使いやすさとデザイン性を両立させたWebアプリケーションの開発を目指しています。

## 開発の目的
* 自身のスキル（HTML5/CSS3/JavaScript/PHP/SQL）の証明と発信
* 制作実績のアーカイブおよび自己紹介
* 今後のエンジニアとしてのキャリア形成

## 使用技術 (Tech Stack)
* **Frontend:** HTML5, CSS3 (Flexbox), JavaScript
* **Backend:** PHP
* **Database:** MySQL (SQL)
* **Tools:** Cursor (Editor), Figma (UI Design), Adobe Illustrator (Graphic Design)
* **Hosting:** GitHub Pages

## 主な機能
* **Works:** 過去に制作したロゴやWebサイトの実績紹介
* **Bio:** 経歴と、教育・サービス業での10年間の経験で得た強み
* **Contact:** お問い合わせフォーム

## 制作のポイント
* **デザイン:** FigmaとIllustratorを用いて、わかりやすいWEBサイトになるように設計しました。
* **保守性:** メンテナンスしやすいHTML/CSS構造を意識しています。
